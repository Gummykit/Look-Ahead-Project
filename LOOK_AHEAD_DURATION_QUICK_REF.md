# 🚀 Look Ahead Duration - Quick Reference

## What's New?

When creating a new project, you now enter the **duration in weeks** instead of picking an end date manually.

---

## How to Use

### Creating a Project

1. **Fill Basic Info**
   - Project Name
   - Company Name
   - Location
   - Description (optional)

2. **Pick Start Date**
   ```
   📅 Start Date  01 Jan 2025  ▼
   ```

3. **Enter Look Ahead Duration**
   ```
   Look Ahead Duration (weeks) *
   [−]  [13 weeks]  [+]
   13 weeks = 91 days
   Ends: 01 Apr 2025
   ```
   
   **Ways to enter:**
   - Click "+" to increase weeks
   - Click "−" to decrease weeks
   - Type directly (1-52 weeks)
   - See calculated end date update instantly

4. **Create Project** → Done! ✅

---

## Duration Conversion

| Weeks | Days | Months |
|-------|------|--------|
| 1 | 7 | ~2 weeks |
| 2 | 14 | ~3 weeks |
| 4 | 28 | ~1 month |
| 6 | 42 | ~1.5 months |
| 8 | 56 | ~2 months |
| 13 | 91 | ~3 months |
| 26 | 182 | ~6 months |
| 52 | 364 | ~1 year |

## Quick Buttons

| Action | Result |
|--------|--------|
| Click "−" | Decrease by 1 week |
| Click "+" | Increase by 1 week |
| Type number | Set exact weeks |
| Clear & type | Replace duration |

---

## Constraints

✅ **Minimum**: 1 week  
✅ **Maximum**: 52 weeks  
✅ **Default**: 13 weeks  

Trying to enter outside range? You'll get an error message.

---

## Tips

- 💡 Use round numbers (4, 13, 26, 52 weeks)
- 💡 Duration automatically calculates end date
- 💡 You can always edit end date later in project editor
- 💡 Think in weeks, not exact dates
