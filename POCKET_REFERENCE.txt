## 🆘 TROUBLESHOOTING

### No logs when I tap buttons?

Try:
```bash
npm start -- --reset-cache
```
Then press `r` in terminal

### Logs show but UI didn't change?

Press `r` in terminal to reload

### Buttons not visible?

- Scroll up
- Rotate device to landscape
- Tap near the top of screen

### Still stuck after 5 minutes?

Check: `DEBUG_GUIDE.md` (in project folder)

## ⚡ COMMON FIXES

| Issue | Fix |
|-------|-----|
| No logs | `npm start -- --reset-cache` then `r` |
| UI didn't change | Press `r` |
| Buttons not visible | Scroll up or rotate device |
| Still logged in after logout | Close and restart app |
| Can't tap buttons | Reload with `npm start -- --reset-cache` |
